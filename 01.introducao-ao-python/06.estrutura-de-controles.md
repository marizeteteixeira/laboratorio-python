#### 5. Estruturas de Controle

Estruturas de controle permitem que o fluxo do programa mude com base em condições específicas.

##### Estruturas Condicionais (if, elif, else)

```python
idade = 18

if idade < 18:
    print("Menor de idade")
elif idade == 18:
    print("Tem 18 anos")
else:
    print("Maior de idade")
```
