#### 5. Estruturas de Repetição

Estruturas de repetição permitem executar um bloco de código várias vezes.

##### Laço `for`

```python
# Iterando sobre uma lista
frutas = ["maçã", "banana", "laranja"]

for fruta in frutas:
    print(fruta)
# Saída:
# maçã
# banana
# laranja
```

##### Laço `while`

```python
# Laço while
contador = 0

while contador < 5:
    print(contador)
    contador += 1

# Saída:
# 0
# 1
# 2
# 3
# 4
```
